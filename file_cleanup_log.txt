```
mci_paper/

├── .github/                 # GitHub Actions
├── .gitignore              # Git 무시 파일
├── .gitmodules             # Git 서브모듈
├── .venv/                  # Python 가상환경
├── archive/                # 아카이브된 파일들

├── logs/                   # 로그 파일들
├── main.py                 # 메인 실행 스크립트
├── output/                 # 생성된 콘텐츠
├── PRD.md                  # 제품 요구사항 정의서
├── README.md               # 프로젝트 설명서
├── requirements.txt        # Python 의존성
├── scripts/                # 핵심 스크립트들
│   ├── analyzer.py
│   ├── blog_generator.py
│   ├── categorizer.py
│   ├── database.py
│   ├── db_manager.py
│   ├── logger.py
│   ├── pubmed_crawler.py
│   └── pubmed_parser.py
└── site/                   # Hugo 블로그 사이트
    ├── content/
    ├── hugo.yaml
    ├── public/
    └── themes/
```

## 1. 프로젝트에 필요한 파일들 (보존)

### 핵심 프로젝트 파일
- .git/ : Git 버전 관리
- .github/ : GitHub Actions 워크플로우
- .gitignore : Git 무시 파일 설정
- .gitmodules : Git 서브모듈 설정
- .venv/ : Python 가상환경
- requirements.txt : Python 패키지 의존성
- main.py : 메인 실행 스크립트

### 데이터 및 설정
- data/ : 데이터베이스 파일
- config/ : 설정 파일들
- logs/ : 로그 파일들
- output/ : 생성된 블로그 포스트

### 스크립트 파일 (scripts/)
- analyzer.py : 트렌드 분석기
- blog_generator.py : 블로그 생성기
- categorizer.py : 논문 분류기
- database.py : 데이터베이스 모델
- db_manager.py : 데이터베이스 관리자
- logger.py : 로깅 설정
- pubmed_crawler.py : PubMed 크롤러
- pubmed_parser.py : PubMed 파서
- __pycache__/ : Python 캐시 (자동 생성)

### Hugo 사이트 (site/)
- content/ : 블로그 콘텐츠
- hugo.yaml : Hugo 설정
- public/ : 빌드된 사이트
- themes/ : Hugo 테마

### 현재 문서
- PRD.md : 제품 요구사항 정의서
- README.md : 프로젝트 설명서
